# 🐍 Snake Game (Python Turtle)

A simple Snake Game built using Python's `turtle` module.

## 📌 Features

- Snake movement using keyboard.
- Food appears at random positions.
- Snake grows after eating food.
- Score is displayed at the top.
- Collision detection:
  - Wall collision
  - Snake body collision
- Game Over message.

---

## 🎮 Controls

| Key | Action |
|-----|--------|
| A | Turn Left |
| D | Turn Right |

---

## 🛠 Requirements

- Python 3.x
- turtle (built into Python)

No external libraries are required.

## ⚙️ Game Logic

- The snake starts with **3 segments**.
- Food appears at a random location.
- Every time the snake eats food:
  - Score increases by **1**
  - Snake grows by one segment.
- The game ends if:
  - The snake hits a wall.
  - The snake collides with its own body.
